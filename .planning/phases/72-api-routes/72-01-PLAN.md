  <name>Task 2: Create workflow detail and update endpoint</name>
  <files>src/app/api/workflows/[id]/route.ts</files>
  <action>
Create Next.js App Router dynamic route with:

**GET /api/workflows/:id** - Get workflow with details
- Get authenticated user from Supabase auth
- Parse workflow ID from URL params
- Call `getWorkflowWithDetails(workflowId)` from workflow-db
- Verify user owns the workflow (user_id matches)
- Return WorkflowRunWithDetails object
- Return 404 if not found, 403 if not owner

**PATCH /api/workflows/:id** - Update workflow status/metadata
- Get authenticated user, verify ownership
- Validate body: { status?: WorkflowStatus, metadata?: Record<string, unknown> }
- Call `updateWorkflowStatus()` or `updateWorkflowMetadata()` as needed
- Return updated workflow

**DELETE /api/workflows/:id** - Abandon workflow
- Get authenticated user, verify ownership
- Set status to 'abandoned' (soft delete)
- Return 204 No Content

Pattern: Verify user ownership on all operations.
