# Third-Party Notices

This file lists attributions and licenses for any third-party sound packs included in `public/sounds/*`.

- Currently, only `trendzo-default` placeholders are used. These are internal placeholders and not shipped externally.
- If `ryos-like` or any other pack is added, include its license details here (e.g., CC0, CC-BY) and provide proper attribution lines.

Example format:

- Pack: ryos-like
  - Source: <link>
  - License: CC-BY 4.0 — © Original Author
  - Modifications: normalized, trimmed for UI cues
