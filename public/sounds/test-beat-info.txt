Test Beat Audio Files

These audio files can be used for testing the beat detection functionality:

1. test-beat-1.mp3 - Simple 4/4 beat pattern at 120 BPM
2. test-beat-2.mp3 - Hip-hop style beat at 90 BPM

For online testing, we can also use the SoundHelix test file:
https://www.soundhelix.com/examples/mp3/SoundHelix-Song-1.mp3

To test the beat detection functionality:
1. Open the test-beat-sync page
2. Select one of the audio sources
3. Play the audio and use the Beat Sync Controller to detect beats
4. The controller should display the detected beats and allow synchronizing with elements
