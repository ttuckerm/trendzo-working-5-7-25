Beat Sync Controller Verification

How to verify the Beat Sync functionality:

  1. Click the link below to open the Template Editor.
  2. In the Template Editor, look for the Beat Sync Controller in the Timeline panel (it should appear directly in the timeline header).
  3. Add a sound to the template using the Sound button in the toolbar.
  4. The Beat Sync Controller should show the "Detect Beats" button which, when clicked, will analyze the audio for beats.
  5. After analysis, the controller will display the number of detected beats and create sync points for animations.
Note: If the Beat Sync Controller is not visible in the timeline, check the browser console for any errors.

Troubleshooting

If the Beat Sync Controller is not working properly:

  1. Check that a sound is selected in the template.
  2. Ensure the Timeline panel is visible.
  3. Verify that no JavaScript errors are occurring in the browser console.
  4. Try refreshing the page if the controller doesn't appear.
