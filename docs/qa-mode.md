QA Mode Overlay

Enable by appending `?qa=1` to any objective page URL. The overlay loads the Objectives Matrix, resolves the current objective, evaluates DOM/metrics/timing checks, and displays PASS/FAIL with reasons.

Keyboard
- Toggle: focus overlay and press Enter on Hide/Show
- Save Result: persists to `/api/qa/results`

Links
- Open Playwright test (local)
- View Matrix JSON
