## Viral Recipe Book — Quickstart Checklist

1) Engine Room → Operations Center
- Click "Run QA seed" in the bottom Controls Bar.
- You should see a top-of-page green banner with an Audit #.

2) Viral Recipe Book header
- Click "Recompute Now" in the discovery slide-over (open via the Discovery pill).
- You should see a green banner; Discovery Freshness drops and sections fill.

3) Templates
- Open any card; the slide-over shows Examples and an "Export to Studio" button.

4) Analyzer
- Upload a small MP4; after analysis, you’ll see "Export to Studio" and "Open Script Editor" above results and a green banner with Audit #.

5) A/B Test and Validate
- A/B: Create an A/B Test, then Start Test; it moves to Active, then Completed with a Winner badge.
- Validate: Click "Start 10-Prediction Test"; a run card appears, metrics load.

Tips
- Use the right-side "?" Debug Drawer to view each API call and last status.
- Use "Demo Fill" (next to the Discovery pill) for one-click demo data.
