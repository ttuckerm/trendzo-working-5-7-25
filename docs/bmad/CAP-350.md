### CAP-350 — Satisfaction Guardrails

Goal: Reduce regret and clickbait, reward sustainable satisfaction signals, and enforce exposure caps per ALG‑101.

Guardrails
- Regret/clickbait classifier: penalize items with `regret_prob ≥ τ_regret` (τ default 0.18)
- Early drop penalty: penalize items with expected watch time ≤ 10s
- Return visit bonus: bonus when share_prob high and regret low
- Exposure caps: per-user (20/day) and global (1000) caps; mark items as capped

API & Integration
- Guardrails configuration: `GET/POST /api/guardrails/config`
- Guardrails metrics: `GET /api/guardrails/metrics`
- Applied post exploration in `/api/recs/serve`; emits regular events; caps marked in `constraintsApplied`

Observability
- Metrics counters: exposure_capped, clickbait_penalized, early_drop_penalized, return_visit_bonus
- Admin page planned adjacent to calibration

Acceptance
- Regret −30% vs baseline; policy violations ≤ baseline
- No deterioration of AVD/share rate beyond tolerance
