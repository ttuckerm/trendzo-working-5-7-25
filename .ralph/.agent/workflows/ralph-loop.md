# Ralph Loop — Task Execution Workflow

Work through tasks in `fix_plan.md` one at a time.

---

## Step 1: Read Context

1. Read `PROGRESS.md` — what was tried before, what to avoid
2. Read `fix_plan.md` — find the first `- [ ]` line

If no `- [ ]` items remain, say "All tasks complete!" and stop.

---

## Step 2: Do the Task

1. Say what task you're working on
2. Read relevant files before changing anything
3. Make the change
4. Test it

---

## Step 3: Record Result

If it worked:
- Mark `- [x]` in `fix_plan.md`
- Add success entry to `PROGRESS.md`
- Move to next `- [ ]` task

If it failed:
- Do NOT mark it done
- Add failure entry to `PROGRESS.md` with what went wrong
- Try to fix it
- If stuck after 3 attempts, STOP

---

## Step 4: Repeat

Go back to Step 1 and find the next incomplete task.
