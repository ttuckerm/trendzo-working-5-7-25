The Nine Attributes Framework for Viral Content
Core Principle: The Dual-Bucket System
The entire framework rests on understanding that viral content success depends on two fundamental buckets:
Bucket 1: Getting Attention (4 Attributes)
How do we get content in front of more people initially?
Bucket 2: Holding Attention (5 Attributes)
How do we keep people watching once they start?
The Algorithm's North Star: User Minutes
The Master Equation
User Minutes = Number of Users × Minutes per Session
This drives everything. The algorithm's sole purpose is maximizing user minutes to increase ad revenue. Your content must serve this goal.
The Two Levers
1.	Shares → Bring new users to platform → Increases "Number of Users"
2.	Watch Time (AVD) → Keep users watching → Increases "Minutes per Session"
Part 1: The Four Attributes for Getting Attention (Maximizing Shares)
Attribute 1: TAM Resonance
Definition: Total Addressable Market - How big is the potential audience for this content?
Implementation Framework:
TAM Score = (Niche Size) × (Topic Universality) × (Platform Fit)
Examples:
•	High TAM: Basketball content (30M+ interested users)
•	Low TAM: Fly fishing content (8M interested users)
Actionable Tactics:
•	Choose topics with broad appeal within your niche
•	Layer niche expertise onto universal human experiences
•	Use broad hooks that funnel to specific value
Attribute 2: Sharability Factor
Definition: How likely is someone to share this with others?
The Sharability Triggers:
1.	Identity Reinforcement: "This represents who I am"
2.	Social Currency: "Sharing this makes me look good"
3.	Practical Value: "This will help someone I know"
4.	Emotional Resonance: "This made me feel something strong"
5.	Surprise Element: "I didn't expect that"
Sharability Checklist:
•	Does it make the sharer look smart/funny/caring?
•	Is there a clear person they'd want to send this to?
•	Does it challenge common assumptions?
•	Is there an "aha" moment worth discussing?
Attribute 3: Hook Strength
Definition: The immediate impact in the first 1-3 seconds that stops the scroll
The Hook Trinity (All three must work together):
1.	Visual Hook: What they see (movement, contrast, surprise)
2.	Verbal Hook: What they hear (provocative statement, question)
3.	Written Hook: What they read (bold text, compelling promise)
Hook Formulas That Work:
•	Result First: "I went from X to Y in Z days"
•	Challenge: "Is it possible to..."
•	Contradiction: "Everyone thinks X, but actually Y"
•	Authority: "After analyzing 1000 videos, I found..."
•	Story: "Something weird happened when..."
Attribute 4: Format Innovation
Definition: The visual and structural presentation that maintains engagement
Engagement Hierarchy (Ranked by performance):
1.	Rapid Change Format: New angle/scene every 2-3 seconds
2.	Demonstration Format: Showing while telling
3.	Comparison Format: Side-by-side contrast
4.	Story Arc Format: Beginning, middle, end structure
5.	List Format: Numbered/structured information
6.	Talking Head: Static presentation (lowest engagement)
Part 2: The Five Attributes for Holding Attention (Maximizing Watch Time)
Attribute 5: Value Density
Definition: Amount of useful information per second of content
The Value Equation:
Value Density = (Actionable Insights / Video Length) × Specificity
Implementation:
•	Remove ALL filler words
•	Start with the most valuable point
•	Use specific numbers, not generalities
•	Every sentence must justify its existence
Attribute 6: Pacing & Rhythm
Definition: The speed and flow that keeps viewers engaged
Pacing Framework:
•	0-3 seconds: Hook (fastest pace)
•	3-10 seconds: Context setting (moderate pace)
•	10-40 seconds: Core value (steady pace)
•	40-60 seconds: Payoff + CTA (accelerating pace)
Rhythm Techniques:
•	Pattern interrupts every 3-5 seconds
•	Tonal variations to maintain interest
•	Strategic pauses for emphasis
•	Building toward climax
Attribute 7: Curiosity Gaps
Definition: Open loops that must be closed
The Loop System:
1.	Open Loop: Pose question/tease outcome
2.	Build Tension: Provide context without answer
3.	Delay Satisfaction: Give partial information
4.	Close Loop: Deliver the payoff
Examples:
•	"The third tip is the most important" (must watch all three)
•	"But first, you need to understand why this fails" (context before solution)
•	"The result surprised even me" (outcome teasing)
Attribute 8: Emotional Journey
Definition: The feeling progression that keeps viewers invested
Emotional Arc Options:
1.	Problem → Struggle → Solution → Triumph
2.	Curiosity → Discovery → Surprise → Satisfaction
3.	Skepticism → Evidence → Conversion → Conviction
4.	Confusion → Clarity → Confidence → Action
Attribute 9: Clear Payoff
Definition: The valuable conclusion that justifies the watch time
Payoff Components:
•	Specific Takeaway: What exactly they learned
•	Implementation Path: How to use this information
•	Success Vision: What's possible with this knowledge
•	Next Step: Clear call-to-action

The Nine-Point Quality Check
Before publishing any content, score each attribute (1-10):
1.	TAM Resonance: How broad is the potential audience?
2.	Sharability: How likely are people to share this?
3.	Hook Strength: How powerful is the opening?
4.	Format Innovation: How engaging is the presentation?
5.	Value Density: How much value per second?
6.	Pacing: How well does it maintain momentum?
7.	Curiosity Gaps: How compelling are the open loops?
8.	Emotional Journey: How satisfying is the arc?
9.	Clear Payoff: How valuable is the conclusion?
Minimum Viable Score:
•	Total: 63/90 (7 average per attribute)
•	No attribute below 5
•	At least three attributes at 8+
Performance Prediction Model
Based on attribute scores:
•	80-90 total: Viral potential (1M+ views)
•	70-79 total: Strong performance (100K-1M views)
•	60-69 total: Solid performance (10K-100K views)
•	Below 60: Needs improvement
Part 4: Advanced Optimization Strategies
The Multiplication Method
When you find a winning combination of attributes:
1.	Document the exact attribute scores
2.	Templatize the structure
3.	Vary only 1-2 attributes per iteration
4.	Scale what consistently scores 70+
Platform-Specific Adaptations
Short-Form (TikTok, Reels, Shorts)
•	Prioritize Attributes 3, 4, 6 (Hook, Format, Pacing)
•	15-60 second sweet spot
•	Vertical format mandatory
Long-Form (YouTube)
•	Prioritize Attributes 1, 5, 7 (TAM, Value, Curiosity)
•	8-12 minute sweet spot
•	Horizontal format optimal
Mixed Platforms (Instagram, Facebook)
•	Balance all nine attributes equally
•	Test multiple lengths
•	Adapt format to placement
The Testing Protocol
Week 1-2: Baseline
•	Create 10 pieces of content
•	Score each on all 9 attributes
•	Track performance metrics
Week 3-4: Optimization
•	Identify highest-performing attribute combinations
•	Create variations focusing on weak attributes
•	A/B test improvements
Week 5-6: Scale
•	Double down on 70+ scoring combinations
•	Create templates from winners
•	Systematize production
