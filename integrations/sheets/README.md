# Trendzo Google Sheets Pack

Install
- Open Extensions → Apps Script; paste `Trendzo.gs`.
- Run onOpen() or reload the sheet. Use the Trendzo menu → Setup; paste your API key and base URL.
- Use Pull Leaderboard to fetch data; use Push Conversions Test to send a sample conversion.

Security
- API key sent as X-API-Key. Ensure your key has access to templates:read and pixel:write scopes.
